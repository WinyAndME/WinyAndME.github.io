关于本站

这是一个特别的家庭生日纪念网站。

从不可能，变成1000行代码，也许只需要灵机一动呢。

本站完全采用纯前端实现。没有后端服务器，不存在受攻击风险。前端数据静态存储在世界最大代码开源社区 Github 。感谢网站的技术支持。

由于前端开发100%自学完成，界面设计有诸多不尽人意之处，甚至可能会有奇妙的BUG，敬请谅解。

底部的音乐播放器提供了多首精选曲目，真的可以播放哦！

其余的功能就请自行探索吧！

感谢访问，希望这个简单的网站能留下温暖和美好的回忆。
