关于本站

这是一个特别的家庭网站。

从不可能，变成1000行代码，也许只需要灵机一动呢。

本站完全采用纯 前端 实现。没有后端服务器，不存在受攻击风险。前端数据静态存储在世界最大代码开源社区 Github 。感谢网站的技术支持。

原本的计划是购买域名，这样网址能再短而好记一点，叫winy.online之类的。一年也就需要几十块钱。但是考虑到目前网站还处于展示站阶段，没有什么数据存储和复杂交互，以目前的技术力买域名还是有点小题大做了。

考虑到个人隐私问题，真实姓名和正脸照片在发布版被隐去。完整版保存在本地，并未发布到互联网。

由于前端开发 100% 自学完成，界面设计有诸多不尽人意之处，甚至可能会有奇妙的BUG，敬请谅解。

底部的音乐播放器提供了多首精选曲目，真的可以播放哦！

其余的功能就请自行探索吧！

感谢访问，希望这个简单的网站能留下温暖和美好的回忆。
